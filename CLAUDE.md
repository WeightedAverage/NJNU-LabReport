# 电子专业实验报告生成模板

本文件夹是通用实验报告生成模板，适用于电子专业所有课程（数字电子技术、模拟电子技术、单片机原理、嵌入式系统等）。

## 使用方法

### 方式一：使用 Claude Code Skill（推荐）

1. 复制整个文件夹到实验目录
2. 把实验讲义放到 "05-实验讲义/" 或编辑 "实验信息.txt"
3. 在终端运行 `claude` 进入交互模式
4. 输入：`开始写实验报告`
5. Claude 会自动读取讲义并生成报告

### 方式二：使用 Python 脚本

1. 复制整个文件夹到实验目录
2. 把实验讲义放到 "05-实验讲义/" 或编辑 "实验信息.txt"
3. 在终端运行：`python generate_report.py`
4. 选择生成方式：
   - 方案一：Word报告（自动填充模板）
   - 方案二：Markdown大纲（手动复制到模板）

## 两种生成方案

### 方案一：生成Word报告
- 读取 "01-模板/" 中的学校模板
- 保留原模板格式，填充完整内容
- 表格使用真实Word表格（doc.add_table()）
- 图片位置标注【此处插入xxx图】
- 保存到 "02-生成的报告/"

### 方案二：生成Markdown大纲
- 生成详细大纲，包含所有内容框架
- 数据表格留空，方便填写
- 保存到 "04-大纲草稿/"
- 用户可直接复制内容到模板，或手写

## 已知问题与解决方法

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 编码显示乱码 | Windows终端编码与Python输出不匹配 | 已添加 `sys.stdout.reconfigure(encoding='utf-8')` |
| 分页符错误 | 使用数字7而非枚举常量 | 已改为 `WD_BREAK.PAGE` |
| 样式名称不存在 | 模板没有"Heading 1"等标准样式 | 使用Normal样式+加粗字体 |
| 内容只能追加到末尾 | python-docx的add_paragraph()限制 | 需要XML操作（见下方） |
| 使用伪表格 | 用文本符号模拟表格 | 使用 `doc.add_table()` 创建真实Word表格 |

## Word报告格式要求

### 序号重置规则
- 每个大节（一、二、三...）重新从"一"开始
- 节内小节序号连续（1. 2. 3...）

### 表格生成要求
- 必须使用 `doc.add_table(rows, cols)` 创建真实Word表格
- 禁止使用伪表格（用 `|---|---|` 等文本符号模拟）
- 表头需加粗，数据行正常字体

### 高级插入（需要XML操作）
- python-docx的add_paragraph()只能追加到文档末尾
- 需要在指定位置插入时，使用XML的addnext()方法
- 参考代码：`para._element.addnext(new_para._element)`

## 讲义读取
- PDF: 使用PyPDF2或pdfplumber提取文本
- Word: 使用python-docx读取
- 图片: 使用OCR或让用户手动描述内容
